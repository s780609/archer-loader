# archer-loader
## description
a simple javascript loader

## Install
`npm install archer-loader`

## method
show() => show the loader
parameter: 
  color: [color code]
  speed: [number]
  text: [string]
cloase() => cloase the loader

## Useage
first, import loader
`import loader from 'archer-loader';`
then use show() and close() methods
`loader.show()`
`loader.close()`
